## Prerequisites
1. Clone the repo
   ```sh
   git clone https://github.com/conv22/NinjaSushi.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your Firebase config in `firebase.ts`
4. Run `npm start` in root directory
